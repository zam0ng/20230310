🚩react 설치!!

🚩react router 설치!!

-> npm install react-router-dom@6

🚩redux 설치 !!

-> npm install redux

🚩react 에서 redux를 쓸 수 있게 도와주는

-> npm install react-redux

-> npm install redux-devtools-extension

🚩리액트 css 설치

-> npm i styled-components

-> npm i redux-thunk
